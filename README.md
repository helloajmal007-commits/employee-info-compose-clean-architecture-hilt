# 🚀 Employee Info Demo-App - Clean Architecture + Jetpack Compose + Hilt

A modern **Employee Information Demo App** built with **Kotlin**, **Jetpack Compose**, and **Hilt**, following **Clean Architecture** principles. This project is специально designed for **fresh developers and beginners** to learn how to build scalable, maintainable, and production-ready Android applications.

---

## ✨ Features

* 🧱 Clean Architecture (Presentation, Domain, Data layers)
* 🎯 MVVM (Model-View-ViewModel) pattern
* 👨‍💼 Employee Management (Add, Edit, Delete, View)
* ⚡ Jetpack Compose UI (Modern declarative UI)
* 💉 Dependency Injection using Hilt
* 🔄 State Management with Kotlin Flow / StateFlow
* 🗂️ Scalable & professional project structure
* 🧪 Easy to understand and extend
